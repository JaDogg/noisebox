### Installation and use

```shell
git clone git@github.com:JaDogg/noisebox.git
cd noisebox
pip install -r requirements.txt
python noisebox.py
```
* Optionally Create a Virtual Environment

### Creating an .exe in Windows

```shell
rem this uses build.cmd
build 
```
